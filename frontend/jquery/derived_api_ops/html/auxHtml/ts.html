Timeline Of Specific Target Currency respect to Base Currency

URL:

http://localhost:9001/frank/excel/d/ts

TYPE:

POST

Payload Example:

{"currency": "USD", "dateOnSet": "1999-01-01", "dateEnd": "1999-02-01", "target": "EUR"}

Description:

This end point gives the historical timeline of a target currency respect to base currency.

Base Options

Pick Date For Timeline Onset

Pick Date For Timeline End

Pick Target
